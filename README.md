# Dataset Kasus Narkotika - PN Lamongan

Dataset ini berisi **50 dokumen putusan pengadilan** yang berasal dari **Pengadilan Negeri Lamongan** dengan jenis perkara **narkotika**. Seluruh dokumen dikumpulkan untuk mendukung penelitian dan pengembangan sistem berbasis **Information Retrieval (IR)** serta **analisis teks hukum** di Indonesia.

Setiap dokumen menggambarkan proses hukum secara lengkap, mulai dari identitas terdakwa, uraian perkara, pertimbangan hakim, hingga amar putusan. Dataset ini dapat digunakan untuk mempelajari pola penulisan putusan, terminologi hukum, serta karakteristik kasus narkotika di tingkat pengadilan negeri.

## Tujuan Penggunaan

Dataset ini ditujukan untuk:

* Mendukung penelitian di bidang **Information Retrieval**
* Menjadi bahan uji coba bagi model pencarian hukum berbasis teks.
* Memberikan gambaran nyata mengenai pola dan konteks hukum dalam perkara narkotika di Indonesia.
